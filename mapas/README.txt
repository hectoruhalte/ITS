= Proceso para generar el fixhero tramos.xml =
=
= 1.- Descargar de openstreetmaps el fichero xml correspondiente. Ejemplo: labaro.xml
=
= 2.- El fichero calles tendrá el nombre de las vías tal y como se denominen en openstreetmaps, junto con el id a guardar para cada vía
=     en el fichero tramos.xml y la altura (por defecto 10).
=
=	Ejemplo:
=		
=		Calle del Lábaro,1,10
=		Plaza de Numancia,2,10
=
= 3.- Ejecutar el script mapear.sh "archivo.xml"
=
=	Ejemplo:
=
=		mapear.sh labaro.xml
=
= Importante: 	OJO Con los archivos de puntos de calles anteriores!!
=
=		DE MOMENTO DE TRAMO EN TRAMO, de uno en uno!!
